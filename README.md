# vue_shop

## 简介
  功能：      电商后台管理系统（前端部分）
  主要技术：  Vue + Element UI + axios
  开发工具：  Vue-cli + VSCoder
  心得：      
    1、使用 Elements UI 作为前端开发的组件库，确实很方便。
    2、前端，其实就是调用接口API，然后将从数据库中获取到的数据有机地渲染到页面上。
      关键点：页面布局 和 解决问题的分析思路。
  建议：在技术扎实的基础上，要学会业务分析。
